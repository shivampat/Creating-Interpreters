# Crafting-Interpreters

This code was created by following the instructions listed in [Crafting Interpreters](https://www.craftinginterpreters.com/). 

Currently, I am working on the Java implementation of the Lox programming language.

To compile, run
```
java -d bin/ src/com/shivam/lox/*.java
java -cp bin/ com.shivam.lox.Lox
```

Later, I will implement a C++ version of Lox that requires compiliation to bytecode and execution through a virtual machine.
